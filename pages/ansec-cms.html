Export & Deploy
Download your content as JSON or CSS variables to apply changes to the live site.
This CMS is client-side only. Export your content as JSON to save externally or share with a developer. The CSS Variables export generates a file you can include in your site to apply text changes via var() references.
JSON Export

Downloads all CMS data as a structured JSON file. Save this as your master content backup. Reload it at any time using Import JSON.

JSON Import

Load a previously exported JSON file to restore your content. This will overwrite current CMS data.

CSS Variables

Generates a ansec-content-vars.css file with all text content as CSS custom properties. Include this in your HTML <head>.

Copy as JSON

Copy the full content data object to clipboard. Paste directly into a ansec-content.js file or your build script.
